# MoodJournalCalendar - 心情日记日历

这是一个基于 HarmonyOS ArkTS 开发的心情日记应用，帮助用户记录每天的心情、笔记和标签，并通过日历和图表进行回顾。

## ✨ 功能特性

### 1. 📅 心情日历
- **日历视图**：直观展示每月的心情记录，支持月份切换。
- **补录功能**：支持点击历史日期补录心情（限制未来日期不可补录）。
- **今日心情**：首页快捷入口记录当天心情。
- **空状态引导**：无记录时显示温馨引导插画。

### 2. 🏷️ 标签与搜索
- **热门标签**：首页展示高频使用的标签（Top Tags）。
- **标签筛选**：点击搜索按钮进入筛选页面，支持按标签查看历史日记。

### 3. 📊 数据统计 (Stats Dashboard)
- **实时更新**：统计数据随日记记录实时刷新。
- **周趋势**：基于心情分值（狂喜=100 ~ 愤怒=20）的可视化柱状图。
- **月度分布**：展示当月心情占比。
- **最佳/最差**：自动统计当月心情最好和最差的一天。

### 4. 📝 日记编辑
- **心情选择**：5种心情状态（🤩 狂喜、😊 开心、😐 平淡、😔 难过、😡 愤怒）。
- **详细记录**：支持添加文字备注和自定义标签。

## 🛠️ 技术架构

- **语言**：ArkTS (TypeScript)
- **框架**：HarmonyOS API 12+ (Stage 模型)
- **UI**：ArkUI (声明式开发)
- **导航**：Navigation + NavPathStack
- **数据管理**：单例 Service + 观察者模式 (Observer Pattern) 实现跨组件数据同步。

## 🚀 运行与调试

1. 使用 **DevEco Studio** 打开项目。
2. 确保 SDK 版本匹配 (HarmonyOS NEXT)。
3. 点击 `Run` 运行至模拟器或真机。
4. **验证功能**：
   - 在首页点击 "+" 记录今日心情。
   - 保存后，首页日历应立即显示对应心情图标。
   - 切换到 "统计" 页，查看图表是否更新。
   - 点击首页 "🔍" 按钮，测试标签筛选功能。

## 📋 常见问题

- **Q: 为什么点击未来日期无法补录？**
  - A: 为了保持日记的真实性，系统限制了只能补录过去和今天的日期。

- **Q: 统计图表如何计算分值？**
  - A: 狂喜(100) > 开心(75) > 平淡(50) > 难过(25) > 愤怒(0)，计算每日平均分。
